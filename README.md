## What is Day Trader app?

Day Trader is an attempt to build a trading app to can be plugged into different platforms and have different strategies


**Perfect for:**
- Syncing day trading transactions from brokerage platform to local sql database  
- Journaling of transactions 
- Day Trading /Scalping on multiple time frames with multiple stock symbols 
- Back testing / live testing / replays
- Incorporate multiple indicators at time of purchase and sale for analysis purposes

> **Note:** This project is actively under development. Some features are still in progress and we welcome contributions and PRs!
